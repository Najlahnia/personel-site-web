Déscription du projet

Le répertoire de ce site contient les principales pages HTML suivantes :

1. Une page d'accueil qui contient le titre de votre site web, le menu de navigation du site, une succincte biographie et éventuellement des icônes de lien vers mes comptes de réseaux sociaux.

2.Une page qui décrit mon parcours académique avec mes diplômes obtenus y intégrer des images pour les diplômes.

3. Une page qui décrit mes compétences informatiques maîtrisées (environnements de développement, frameworks, langages de programmation, SGBD, méthodes agiles, etc.).

4. Une page qui contient mes expériences professionnelles.

5. Une page qui contient les manifestations scientifiques, les journées de formation, les évènements dont j'ai participés y intégrer des photos de ces manifestations.

6. Une page qui contient les matières que vous avez étudiées et éventuellement mes TP/projets réalisés.

7. Une page qui décrit mes passions, mes sports préférés et mes qualités.

8. Une page qui contient un quizz (un formulaire HTML). Le visiteur du site web est amené à répondre à ce quizz et il aura sa note une fois qu'il clique sur le bouton de validation. Cette note est accompagnée par une liste des réponses correctes.

9. Une page de contact : un formulaire HTML qui permet de m'envoyer un email.

10. Une page de témoignage sur l'IIT.
